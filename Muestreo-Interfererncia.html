Inferencia¶

Para este caso comparamos 3 variables; Razonamiento cuantitativo, ingls, Lectura Crtica contra a una cuarta creada para entrar en contexto la cual es "Puntaje Global" definida como la suma de todas las variables cuantitativas divididas por 5

Las grficas las creamos tomando 3 tamaos de muestra distintos el primero N1 = 26, N2=102 y N3 = 1581, variando los niveles de confianza en 90%, 95% y 99% y el error en 5%, 3% y 1%.

Podemos apreciar que el primer y segundo muestreo, resultaban deficientes al no permitir apreciar con un nivel de detalle adecuado el comportamiento de los datos, mientras que el valor N3 no dista de los resultados obtenidos de la poblacin total a un menor costo computacional.

De los resultados obtenidos podemos apreciar que la asignatura ingles representa un mayor reto para la mayora de los estudiantes teniendo un pico mucho ms pronunciado hacia los 120 pts y decayendo con una pendiente mayor hacia los puntajes ms altos, a diferencia de Razonamiento Cuantitativo y Lectura crtica que pueden ser fcilmente modeladas por una distribucin normal

Inferencia¶

La diferencia principal entre los resultados obtenidos en el Muestreo Aleatorio Simple (MAS) y el Muestreo Estratificado (ME) es que en el primero se escoga una muestra de manera aleatoria sin tener en cuenta ninguna caracterstica de la poblacin, y en el segundo se escogi al periodo como el respectivo estrato a seguir y se logr que todos esos estratos estuvieran representados de manera adecuada en la muestra. Analizando las grficas observamos que en el diagrama de cajas del MAS existe solo una caja mientras que en el del ME hay 7, cada una correspondiente a cierto periodo.

4- Muestreo Sistematico

En primer lugar, debemos tener claro que el muestro sistemtico parte de un nmero aleatorio i y los elementos que integran la muestra son los que ocupa los lugares i,i+k,i+2k,i+3k,...,i+(n-1)k. El riesgo de este tipo de muestreo est en los casos en que se dan periodicidades en la poblacin, ya que al elegir a los miembros de la muestra con una periodicidad constante (k) podemos introducir una homogeneidad que no se da en la poblacin. En este caso en concreto los datos estn ordenados por periodos, siendo que los primeros datos del data set son del 2016, los de en medio 2017 y los del final 2018. Esto nos indica que la aleatoriedad no nos asegurar una distribucin homognea y por el contrario la sistemtica s recorrer el data set de a pasos enteros asegurando nmeros homogneos.

funcion_estimaciones <- function(x,propor_p, na.omit=FALSE){
    
    if (na.omit)
        x <- x[!is.na(x)]
    
    tamao <- sample.size.prop(e=0.05, P = 0.5, N = length(x), level = propor_p)$n 
    muestra <- sample(1:length(x), size = tamao, replace = FALSE)
    
    valores <- x[muestra]
    varianza = round(popvar(valores),2)
    media = round(mean(valores),2)
    
    return(c("media"=media,"varianza" = varianza))
}

funcion_intervalo <- function(variable,valordeZ){
min_x = min(variable)
max_x = max(variable)
std_x = sd(variable)
        
media <- mean(na.omit(variable)) # Pedimos la media
desv <- sd(na.omit(variable)) # La desviacin estndar
N <- length(na.omit(variable)) # El tamao vlido de la muestra
error.est <- desv/sqrt(N) # Calculamos el error estndar
    
error <- valordeZ*error.est 
lim.inf <- media-error # Lmite inferior del intervalo
lim.sup <- media+error # Lmite superior del intervalo
return(c("limite superior" = lim.sup, "limite inferior"= lim.inf))
    
}
